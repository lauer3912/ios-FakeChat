# FakeChat

**Fake Chat Screenshot Generator** — Create realistic fake chat screenshots for iMessage, Telegram, Snapchat, WhatsApp, Instagram DM, Twitter/X, and Discord.

## Features
- 🎭 Create fake conversations in 7 chat apps
- 📸 Generate high-resolution screenshots
- 🎨 Dark neon theme (default) + light theme
- 📁 Templates for quick creation
- 💾 History to save and revisit your creations
- ✨ Pro features: more apps, GIF/video export, no watermark

## Tech Stack
- UIKit (programmatic UI)
- MVVM architecture
- iOS 15.0+

## Privacy
All data is stored locally on device. No analytics, no account required.
Privacy Policy: https://lauer3912.github.io/ios-FakeChat/docs/PrivacyPolicy.html

## Build
1. Open `project.yml` with XcodeGen
2. Generate project with `xcodegen generate`
3. Build and run on simulator
